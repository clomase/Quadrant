Лёгкий оркестратор виртуальных машин на базе чистого QEMU — без необходимости использования `sudo`. Предназначен для ограниченных сред (учебные лаборатории, корпоративные рабочие станции), где отсутствует доступ к правам администратора.

## О проекте

Quadrant позволяет быстро разворачивать кластеры из нескольких ВМ через QEMU без административных привилегий:

 - **Без прав суперпользователя** — работает полностью от обычного пользователя с использованием пользовательского режима сети QEMU
 - **Многоэтапный провижининг** — разделяет базовую настройку (установка Docker) и продвинутые задачи (инициализация Swarm, обмен токенами между нодами)
 - **Внутренние сети через сокеты** — соединение ВМ через пары сокетов `listen/connect` без настройки мостов и `sudo`
 - **Индивидуальная конфигурация каждой ВМ** — настройка CPU, RAM, сетей через файл `Quadrantfile`
 - **Изоляция состояния** — все данные хранятся в `~/.q-drant/`, без загрязнения системных директорий

## Требования

 - Bash 4.3+
 - SSH-клиент (`ssh`, `scp`)
 - QEMU (`qemu-system-x86_64` или бинарник для другой архитектуры)
 - Базовый образ в формате QCOW2 с доступом по SSH
